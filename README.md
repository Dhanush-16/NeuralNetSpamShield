NeuralNetSpamShield is an advanced spam filter developed using supervised learning and neural networks. This project aims to effectively identify and filter out spam messages by leveraging the capabilities of deep learning algorithms.
<h1>Spam Filter</h1>
A Machine Learning web application that detects whether a message is Spam or Ham (Not Spam) using a trained LSTM neural network. The model is deployed using Flask and allows users to test messages through a simple web interface.
